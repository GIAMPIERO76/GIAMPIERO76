ActiveXperts Serial Port Component is an ActiveX/COM component that can be integrated in virtually any Windows development environment. The 32 bit component is a single DLL file and is installed here: The 32 bit component is a single DLL file and is installed here:
    C:\Program Files\ActiveXperts\Serial Port Component\Program\AxSerial32.dll
The 64 bit component is a single DLL file and is installed here:
    C:\Program Files\ActiveXperts\Serial Port Component\Program\AxSerial64.dll
